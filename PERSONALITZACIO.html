🍀 Personalitza Celtic Girona

Benvingut!

Aquesta guia et mostrarà com personalitzar tots els aspectes de la web del Celtic Girona des de les colors fins als continguts.

1️⃣ Canviar Colors i Estils

Obri el fitxer styles.css i trobaràs les variables CSS al principi:

:root {
    --primary: #22c55e;      /* Verd principal - canvia aquí */
    --secondary: #16a34a;    /* Verd secundari */
    --dark: #0f172a;         /* Fons fosc */
    --light: #f8fafc;        /* Fons clar */
    --text: #1e293b;         /* Color text */
}

Pista: Pots usar qualsevol codi de color heptadecimal (hex). Prova https://colorpicker.com

2️⃣ Actualitza Continguts Principals

At index.html:

Logo i nom del club:

<div class="logo">
    <i class="fas fa-clover"></i>
    <span>Celtic Girona</span>
</div>

Slogan principal:

<p class="hero-subtitle">Passió, Tradició i Futur</p>

Pots cambiar l'icona fa-clover (trèvol) per uns altrers:

  • fa-futbol - Pilota de futbol
  • fa-shield - Escut
  • fa-star - Estrella
  • fa-heart - Cor

3️⃣ Secció "Qui Som"

Edita les tres tarjetes:

<div class="card">
    <div class="card-icon"><i class="fas fa-history"></i></div>
    <h3>La Nostra História</h3>
    <p>Aquí va el teu text personalitzat...</p>
</div>

4️⃣ Articles de Premsa

Modifica els articles existents o afegeix-ne de nous:

<article class="article-card">

8️⃣ Afegir Imatges

  1. Crea una carpeta /images si no existeix
  2. Copia les teves imatges allí (logo.png, stadium.jpg, etc.)
  3. Canvia els placeholders per imatges reals: 
    <img src="images/myimage.jpg" alt="descripció">
  4. Redimensiona les imatges (es recomana:
    • Hero image: 600x400px
    • Article images: 400x300px
    • Logo: 150x150px
    )

9️⃣ Animacions i Efectes

Els efectes estàn configurats automàticament! Però pots personalizar-los:

  • Velocitat de scroll reveal: En script.js cerca duration: 0.6s
  • Colors de gradients: En styles.css modifica els linear-gradient
  • Parallax intensitat: Canvia el multiplicador en scrolled * 0.5

🔟 Publicar la Web

Opció 1: Netlify (Recomanat)

  1. Crea un compte a netlify.com
  2. Connecta amb GitHub
  3. Selecciona els fitxers de la carpeta
  4. Click en "Deploy" - llestos! ✨

Opció 2: GitHub Pages

  1. Crea un repositori a GitHub
  2. Puja els fitxers
  3. En Settings → Pages → Select "main" branch
  4. La web estarà en: username.github.io/celticgirona

Opció 3: Hosting Tradicional

  1. Contracta un hosting (GoDaddy, Hostinger, etc.)
  2. Connecta via FTP
  3. Puja els fitxers a la carpeta public_html

Easter Eggs Secret! 🎉

Escriu CELTIC mentre navegues per la pàgina per veure una sorpresa amb confetti:

  • Prova a escriure les lletres lentement: C-E-L-T-I-C
  • Obra la consola del navegador (F12) per veure missatges secrets
  • Mira els efectes al fer scroll
  • Prova els efectes hover en tots els elements

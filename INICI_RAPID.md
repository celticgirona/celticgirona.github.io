## 🌐 Publicar la Web

### Opció 1: NETLIFY (Recomanat ⭐)
1. Entra a https://netlify.com
2. Crea compte gratis
3. Drag & drop la carpeta
4. Done! URL automàtica: `xxxxx.netlify.app`

### Opció 2: GitHub Pages
1. Entra a https://github.com
2. Crea nou repo "celticgirona"
3. Puja els fitxers
4. Settings → Pages → Deploy
5. Done! URL: `username.github.io/celticgirona`

### Opció 3: Hosting Tradicional
1. Contracta hosting (Hostinger, OVH, etc.)
2. Connecta per FTP
3. Puja fitxers a `public_html`
4. Done!

## 🐛 Troubleshooting

### "Veig els fitxers, però la web no es veu"
✓ Assegura't d'obrir `index.html` directament des del navegador

### "Falta fulls me personals o imatges"
✓ Comprova els paths en el codi
✓ Les imatges han d'estar a la carpeta `/images`

### "Els colors no canvien"
✓ Assegura't de guardar `styles.css`
✓ Refresca la pàgina (Ctrl+F5 o Cmd+Shift+R)
✓ Borra cache del navegador

### "El menú no funciona en mòbil"
✓ Proba a recarregar la pàgina
✓ Vifica que `script.js` estava ubicat correctament
